# 关于本站

本站使用VuePress+github搭建部署。VuePress是一款基于Vue.js框架的静态网站生成器，它被设计用于开发技术文档、博客、单页应用等。

🎯VuePress遵循Vue.js的约定大于配置的原则，同时结合了Webpack和Markdown等技术，使得它能够快速生成高效、可靠、易于维护的静态网站。

🍺VuePress支持主题定制和插件扩展，可以通过编写Vue组件和自定义Webpack配置来满足不同的需求。

🍧VuePress的默认主题使用了Vue.js官方风格，并包含了许多有用的特性，例如自动生成的侧边栏、主题切换、搜索功能等。

🏺在使用VuePress生成静态网站时，可以使用Markdown语言编写内容，并通过VuePress提供的页面路由系统和自动生成的导航栏快速引导用户浏览不同页面。

🍖此外，VuePress还内置了一些常用插件，如Google Analytics、PWA支持等，以进一步提高网站的性能和用户体验。

总的来说，VuePress是一个非常灵活和易于使用的静态网站生成器，适用于需要频繁更新或团队协作的技术文档、博客、单页应用等。
VuePress官网：https://v1.vuepress.vuejs.org/ (opens new window)

# 导航网站

🔞导航网站是一种由大量网址链接组成的网站，旨在帮助用户快速方便地找到他们感兴趣的网站。导航网站通常会按照主题或领域进行分类和整理，使得用户可以更轻松地找到相关资源。

㊙️一些导航网站还提供搜索功能和社交化分享功能，使得用户可以通过关键字、标签等方式查找感兴趣的内容，并与其他用户共享自己的收藏夹和网址。

导航网站是一个非常实用的工具，它能够帮助用户快速找到各种资源，大大减少了网上冗余搜索的时间和成本。
