# 代码托管

🍎代码托管平台是指一种服务，提供在线的代码管理功能，并可以帮助团队协作、版本控制和代码发布等功能。它允许开发者将自己的代码存储到云端，而不是在本地进行存储，从而提供了更加便捷高效的代码管理方式。

# 免费图床

🍓图床指的是一种在线图片存储服务，通过这个服务，用户可以将自己的图片上传到云端，获取一个图片链接，然后在需要使用这些图片的地方，使用该链接来引用相关图片。

名称 网址
路过图床 https://imgse.com/ (opens new window)
聚合图床 https://www.superbed.cn/ (opens new window)

# 免费音乐外链

🍅音乐外链是指通过某些在线音乐服务，获取一个音乐文件的链接，并将该链接嵌入到自己的博客、网站等页面中，使得访问者可以直接在用户的页面上播放这首音乐。通常，这个链接会使用HTML标签（如<audio>）进行引用，以实现音乐的在线播放。

名称 网址
JOY127 https://www.joy127.com/ (opens new window)
