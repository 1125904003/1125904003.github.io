# 前端开发之HTML篇

# HTML 发展历程

HTML（HyperText Markup Language，超文本标记语言）的发展历程可以大致分为以下阶段：

HTML 1.0：于 1993 年发布，是 Web 技术的第一个正式规范。它包含了最基本的 HTML 元素和属性，并初步定义了 Web 页面的结构和语法。

HTML 2.0：于 1995 年发布，是 HTML 1.0 的升级版，增加了一些新的元素和属性，例如表格<table>和字体<font>等等。

HTML 3.2：于 1997 年发布，引入了一些新的元素和属性，例如层和框架（<layer>、<frame> 和 <iframe>）等，同时也提供了对样式和脚本的支持。

HTML 4.01：于 1999 年发布，是 HTML 3.2 的升级版，修复了一些漏洞和错误，并增加了一些新的元素和属性，例如嵌入式对象<object> 和 <embed>、样式表<style>和表单控件<textarea> 和 <fieldset>等等。

XHTML 1.0：于 2000 年发布，是 HTML 的 XML 版本，以更严谨的语法规则和模块化的结构来定义 Web 页面。XHTML 1.0 中移除了一些旧的元素和属性，同时也增加了对样式和脚本的支持。

HTML5：于 2014 年正式成为 W3C 推荐标准，是当前 Web 技术中最新的 HTML 标准，增加了多种新的元素、属性和 API，包括语义化元素、媒体元素扩展、表单控件扩展、Canvas 元素、地理定位 API 等等。同时，HTML5 还提供了更好的兼容性和跨平台支持，使得开发 Web 应用程序变得更加易于操作和高效。

# HTML 常见标签

HTML（Hypertext Markup Language）是一种用于创建网页和其他 Web 应用程序的标记语言。它不是一种编程语言，而是一种标记语言，因此它不具有编程语言的功能，例如条件分支和循环等。HTML 由一堆标签（Tag）和文本内容组成，这些标签用于描述网页文档的各个部分，并决定浏览器如何渲染这些部分的样式和行为。

下面是 HTML 的一些基本元素：

<!DOCTYPE html> 声明：用于声明文档类型。

<html> 元素：用于定义一个 HTML 文档。

<head> 元素：包括一些定义文档信息和链接到其他文件的元数据，例如 title、link 和 script 等。

<body> 元素：包含可见的页面内容，例如文本、图像、超链接、表格等。

标题<h1-h6>元素：用于设置文档的标题级别，从 h1 到 h6 共 6 级，级别递减。

段落<p>元素：用于在文档中标记段落，并在网页上显示相应的格式化文本内容。

超链接<a>元素：用于在文档中加入一个可点击的链接。

图像<img>元素：用于在文档中嵌入图像。

列表<ul、ol、li>元素：包括有序列表（ol）和无序列表（ul），用于创建有序或无序的列表。

表格<table>元素：使用表格标签（table）可创建一个表格，使用行标签（tr）、表头标签（th）、单元格标签（td）等等来表示表格中的行、列和单元格等内容。

表单<Form>元素：用于创建 Web 应用程序中的表单，包括文本输入框、密码框、复选框、单选框和下拉列表框等。

视频<Video>元素和音频<Audio>元素：用于在网页上播放视频和音频内容。

文本格式化元素：包括粗体<b> 或 <strong>、斜体<i> 或 <em>、下划线<u>等等。

划分页面结构的元素：包括 <div> 和 <span> 等，它们通常用于将页面划分为独立的部分，以便于对其进行样式定义和布局操作。

脚本<Script>元素：用于嵌入 JavaScript 代码，实现动态效果和交互。 框架<Frame>元素：用于在一个页面中嵌入另一个页面。

# HTML5 新特性

语义化元素：HTML5 引入了一些语义化的元素，例如 <article>、<section>、<header>、<footer> 等等，它们有助于开发人员更好地描述页面内容的结构和含义，也可以提高搜索引擎对页面内容的理解和处理。

媒体元素扩展：HTML5 引入了 <video> 和 <audio> 元素，这两个元素支持多种格式的媒体文件，包括 MP4、WebM 和 Ogg 等等。此外，HTML5 还支持通过 <source> 元素提供多种格式的媒体文件，以便浏览器选择最适合的一个来播放。

表单控件扩展：HTML5 引入了一些新的表单控件和属性，例如日期选择器（<input type="date">）、范围选择器（<input type="range">）、颜色选择器（<input type="color">）等等。

Canvas 元素：HTML5 引入了 <canvas> 元素，它提供了一种使用 JavaScript 在网页上绘制图形和动画的方式。

地理定位 API：HTML5 引入了 Geolocation API，用于获取用户位置信息，可以在 Web 应用程序中根据用户位置提供个性化服务。

Web 存储：HTML5 引入了多种 Web 存储技术，包括 Local Storage、Session Storage 和 Web SQL Database 等，它们可以在浏览器本地保存数据，方便离线访问和数据共享。
