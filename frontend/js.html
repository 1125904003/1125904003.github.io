# JavaScript 发展历程

JavaScript是一种广泛应用于Web开发的脚本语言，以下是JavaScript发展历程的主要里程碑：

1995年，Netscape公司的Brendan Eich设计了JavaScript，最初被称为Mocha和LiveScript，后来改名为JavaScript。当时的目标是创建一个简单、易学、互动性强的脚本语言，用于在网页中添加动态效果。

1996年，微软推出了自己的JavaScript版本JScript，并将其纳入IE3浏览器中。从此，JavaScript逐渐成为Web开发的标准脚本语言，被广泛应用于网页交互和动态效果的实现。

1997年，ECMA（European Computer Manufacturers Association）发布了ECMAScript标准，定义了JavaScript的基本语法和规则，并将其作为一种国际标准来推广。

随着Web技术的不断发展，JavaScript也经历了许多变化和演进。其中比较有代表性的版本包括：

JavaScript 1.2：支持正则表达式、数组方法和Date对象等特性。

JavaScript 1.5：支持函数式编程、闭包、原型继承和异常处理等特性。

ECMAScript 3：1999年发布，对语言本身的功能进行了修订和完善。

ECMAScript 5：2009年发布，增加了严格模式、JSON对象和高阶函数等特性。

ECMAScript 6（ES6）：2015年发布，引入了let/const、箭头函数、模板字符串、类和模块等新特性。 目前，JavaScript已经成为Web开发不可或缺的一部分，几乎所有现代浏览器都支持ECMAScript标准，并且越来越多的应用程序开始使用JavaScript进行开发。随着Web技术的不断发展，JavaScript也在不断地完善和扩展其功能，成为了一个功能强大、灵活和易于学习的脚本语言。
ECMAScript 标准文档：https://tc39.es/ecma262/ (opens new window)

# JS简介及相关资源

JavaScript（通常缩写为 JS）是一种高级的、解释型的编程语言，用于在 Web 页面上创建动态交互效果、处理数据、控制浏览器等等。它是现代 Web 开发中不可或缺的一部分。

由 Brendan Eich 在 1995 年首次设计并实现，JavaScript 最初是作为 Netscape Navigator 浏览器的一项扩展而诞生的，现在已经成为了一门广泛应用的编程语言。今天，JavaScript 的应用范围已经扩展到了服务器端、桌面应用程序和移动应用开发等领域。
W3Schools JavaScript 教程：https://www.w3schools.com/js/ (opens new window)
MDN Web Docs JavaScript 文档：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript (opens new window)
