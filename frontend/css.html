# CSS 发展历程

CSS（Cascading Style Sheets）是一种用于描述网页样式和布局的语言。随着Web技术的发展，CSS经历了许多变化和演进，以下是CSS发展历程的主要里程碑：

CSS1：1996年发布，成为一种独立的样式表语言，用于定义文档的外观和排版。

CSS2：1998年发布，增加了对定位、浮动、字体控制、分页和打印布局等方面的支持。

CSS2.1：2004年发布，修正了CSS2中一些错误，并增加了对随机媒体、用户界面和国际化的支持。

CSS3：从2001年开始逐步提出各个模块，包括选择器、盒模型、背景与边框、文本效果、2D/3D转换、动画、多列布局、弹性盒子布局、网格布局等。

CSS4：目前还没有正式发布，但是已经在不断地完善和补充新的特性。其中比较有代表性的是CSS Variables（CSS变量）、Grid Layout（网格布局）等特性。

CSS经历了20多年的发展，从最初的CSS1到现在的CSS4，不断地增加了新的特性和功能，使得Web开发人员可以更好地控制页面的外观和布局，为Web应用程序提供更好的用户体验。
